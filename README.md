# Advent of Code 2025

This repository contains my solutions for the Advent of Code 2025 programming challenges. Each day's challenge is implemented in a separate file, organized by day and part.

## Directory Structure

```bash
scripts/
--- build.sh          # Script to build/compile the solutions
src/
--- logger/         # Logger utility for debugging and logging
--- day01/          # Solutions for Day 1
...
--- day12/         # Solutions for Day 12 (final day)
build/            # Compiled binaries (if applicable)
```
